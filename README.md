# TokTik Back End

## 1. Install Deps
```bash
npm install
```

Please save this `.env` to the root directory:

```
BACKEND_PORT=4000
DB_URL=MONGO_DB_URL_PROVIDED_IN_FORM
```

Remember to replace `MONGO_DB_URL_PROVIDED_IN_FORM` with the link we provided.

## 2. Start
```bash
npm start
```

## Test

```bash
npm test
```

For lint, you can use `npm run lint`.
